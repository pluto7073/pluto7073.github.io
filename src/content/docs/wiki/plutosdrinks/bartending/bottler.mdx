The **bottler** is a block used to finalize and seal an alcoholic drink.

## Obtaining

### Breaking
A bottler can be mined with anything, but [pickaxes](https://minecraft.wiki/w/Pickaxe) are the fastest. When broken, it also drops all of its contents except the display item.

## Usage
First, place the concoction with all the steps completed in the first glass bottle slot.  If the steps done match a recipe for a drink, the arrow will slowly start to fill up.  If not, the display slot
will be changed to a ruined concoction item.

If the steps match, then after the arrow is filled the display slot will be replaced with the finished drink, including any deviation in alcohol content as a
result of differences in the recipe.

Whether the result is a complete alcoholic drink or ruined concoction, you can collect this by placing the correct bottle in the second glass bottle slot, and the result will be placed in the fourth slot.
