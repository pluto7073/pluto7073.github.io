## Usage
> *Main Article: [Drink Add-Ins](https://github.com/pluto7073/PlutosDrinksAPI/wiki/Drink-Add%E2%80%90Ins)*

To use a **Drink Workstation**, simply open the UI using the *use* key.  Then, place the [base item](https://github.com/pluto7073/PlutosDrinksAPI/wiki/Valid-Drink-Bases) in the first slot and the addition item in the second slot, then the result will appear in the 3rd slot.  The item will have a tooltip showing the add-ins and the amount of each.
