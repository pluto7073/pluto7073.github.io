Pluto's Coffee Mod adds coffee and related features into the game

### Want More Drinks?
Check out the entire Pluto's Drinks collection [here](https://modrinth.com/collection/n9TVye64)!

## Features

- Coffee Cherries, Beans, Grounds, and Coffee
- Brew using the Coffee Brewer
- Add in milk, sugar, caramel, and mocha using the Coffee Workstation
- Steam milk and pull espresso shots to create lattes

## Cross Mod Features
- Coffee will hydrate you in the [Dehydration](https://modrinth.com/mod/dehydration) mod
- Lattes support Farmer's Delight Milk Bottles (as of 3.0)
